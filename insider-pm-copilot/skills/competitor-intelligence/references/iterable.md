# Iterable — Competitor Reference

## Overview
Iterable is a cross-channel customer lifecycle marketing platform founded in 2013, headquartered in San Francisco. Backed by Silver Lake (took the company private in 2023 from earlier VC funding). Targets growth-stage to enterprise B2C companies in e-commerce, media, SaaS, fintech, and travel. Known for strong lifecycle marketing capabilities and developer-friendly APIs.

## Positioning
"Iterable is the AI-powered cross-channel marketing platform enabling brands to create joyful customer experiences." Positions as the modern lifecycle marketing platform with strong data and developer foundations. Competes directly with Braze in the enterprise cross-channel space, often positioned as a more flexible/developer-friendly alternative.

## Core Product Areas

### Channels
- Email, SMS, Push Notifications (iOS + Android + Web), In-App Messages, Direct Mail (via partner integrations), WhatsApp (via partnerships), Webhooks, Custom Channels (via API)

### Segmentation
- User properties, events, behavioral, predictive audiences (AI-powered), computed fields. Static and dynamic lists. SQL segments for advanced users.

### Journey Orchestration
- Studio (Workflow builder): visual drag-and-drop journey canvas, multi-channel, event-triggered, time-based, A/B testing, holdout groups, branching logic. Filter steps, webhook steps, delay steps.

### Personalization & AI
- Iterable AI: brand affinity modeling, predictive goals, send-time optimization, intelligent ingest (data quality), generative AI tools (subject line, copy). Catalog-based personalization using product/content catalogs.

### Data & CDP
- Iterable Data Model: user profiles, events, catalogs (product/content). API-first ingestion. Iterable Insights for SQL-based analytics. Integration with CDPs (Segment, mParticle, Hightouch, Census). No native CDP — relies on external data sources.

### Analytics
- Campaign analytics, journey analytics, A/B test reporting, cohort analysis, revenue attribution. Iterable Insights: SQL-based custom analytics (BigQuery under the hood).

### Developer Experience
- REST API (comprehensive and well-documented), iOS SDK, Android SDK, Web SDK, React Native SDK. Custom channel API for building proprietary channels. Webhooks for event streaming. Very developer-friendly.

### Integrations
- 100+ integrations: Segment, mParticle, Snowflake, BigQuery, Amplitude, Mixpanel, Salesforce, Shopify, Magento, AppsFlyer, Braze SDK compatibility via custom channels.

## Strengths
- Developer-friendly API and SDK ecosystem
- Strong lifecycle marketing and B2C cross-channel capabilities
- Studio (workflow builder) is highly flexible and powerful
- Good e-commerce catalog personalization
- Brand affinity AI is genuinely differentiated
- Iterable Insights (SQL analytics) appeals to analytics-heavy teams

## Weaknesses
- Not a native CDP — data model is less sophisticated than Bloomreach or Insider
- Web/onsite personalization is limited (no native web experience builder)
- WhatsApp support is via partnerships, not native
- Mobile in-app capabilities less rich than Braze or MoEngage
- Going private (Silver Lake) creates some uncertainty around investment and roadmap
- Less brand awareness than Braze or Klaviyo in mid-market

## Pricing Model
Contact-based (user profiles) + message volume. Enterprise contracts. Pricing not publicly listed. Generally competitive with Braze for similar-scale deployments.

## Key Differentiators vs. Insider One
- Iterable competes in mid-to-enterprise cross-channel — direct Insider overlap
- Insider One has native web/onsite personalization; Iterable does not
- Insider One's CDP is native and operational; Iterable relies on external CDPs
- Iterable wins with developer-led teams who value API flexibility
- Insider One's product recommendation engine and e-commerce depth is stronger
- Both have strong A/B testing and journey orchestration — close capabilities

## Recommended Talking Points (Insider vs. Iterable)
- "Insider One includes native web and onsite personalization — Iterable can't serve the full customer journey"
- "Insider's built-in CDP eliminates the need to integrate a separate customer data platform"
- "Insider's AI-driven product recommendations are deeply connected to your product catalog — driving direct revenue impact"
- "Insider One's omnichannel coverage (including WhatsApp natively) is more complete"
- "Insider's time-to-value is faster — Iterable's developer-first model requires more engineering investment"
