# Insider One Product Taxonomy

Use this taxonomy to classify PM queries and enrich search keywords. When a query matches a product area, add the associated keywords to your searches across all sources.

If a query does not clearly match a product area, search all sources without taxonomy-enriched keywords.

## Product Areas

| Product Area | Description | Keywords |
|---|---|---|
| Web Suite | On-site personalization, A/B testing, web push notifications, banners, popups, social proof | web, onsite, banner, popup, AB test, web push, social proof, exit intent |
| App Suite | In-app messaging, app push notifications, app personalization, mobile SDK | mobile, in-app, app push, SDK, app personalization, deep link |
| Messaging Channels | Email, SMS, WhatsApp, RCS, LINE, Facebook Messenger | email, sms, whatsapp, rcs, line, messenger, template, campaign |
| CDP | Customer data platform, user segmentation, profile unification, audience management | cdp, segment, audience, unification, profile, attribute, event, identity |
| Architect | Journey orchestration, cross-channel flow builder, automation workflows | journey, architect, flow, orchestration, workflow, automation, trigger, branch |
| AI & Recommendations | Sirius AI, predictive models, product recommendations, send-time optimization | ai, sirius, recommendation, prediction, lookalike, send-time, optimization, ML |
| Analytics | Funnels, cohorts, reporting, dashboards, revenue attribution | analytics, funnel, cohort, report, dashboard, attribution, conversion |
| Integrations | Connectors, APIs, webhooks, data import/export, third-party platforms | integration, api, connector, webhook, import, export, sync |

## Usage

1. Parse the PM's query for keywords matching the table above.
2. If a match is found, note the product area(s) — a query can span multiple areas.
3. Append the matched keywords to your search queries for Confluence, Jira, Academy, and codebase.
4. In your synthesized answer, mention which product area(s) the answer relates to.
